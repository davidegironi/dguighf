About
===

**DGUIGHF** is a **UI GUI Helper Framework** built for *.NET WinForm*.

## Download

+ **[NuGet](https://www.nuget.org/packages/DG.UIGHF)**
+ **[Latest Release](../../releases/latest)**

## Requirements

* Microsoft Windows with .NET framework 4.5.2 or later
